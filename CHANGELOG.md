# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-01

### Added

- Initial release of Directory Structure Validator.
- Support for newline, comma, and space-separated directory lists.
- Configurable base path support.
- Optional non-failing mode with `fail-on-missing`.
- Outputs for validation status, checked directories, and missing directories.
- GitHub Marketplace-compatible action metadata.
